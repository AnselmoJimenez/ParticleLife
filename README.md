# ParticleLife

> [!NOTE]
> This project is a work in progress and is not in it's final state!

Particle Life is a type of artificial life simulation that emerged from explorations in emergent complexity and self-organization.  
The core idea behind Particle Life stems from simple rule-based systems where particles interact through attraction and repulsion forces. The concept builds on earlier work in artificial life and complexity science, particularly cellular automata and agent-based modeling.
